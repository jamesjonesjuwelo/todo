TO-DO

To initialise the application, run:
1. yarn install
2. yarn build
3. yarn start
4. create a local PostgreSQL db, named 'listdb'

The project will then be running via http://localhost:8080/, with the database running on port 5432.

Technologies used
1. React
2. Redux
3. Redux Observables
4. Styled-Components
5. Axios
6. Jest
7. Enzyme
8. Node.js
9. Express
10. PostgreSQL
11. Cookie Sessions
